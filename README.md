# Teslo API
1. Clonar proyecto
2. ```yarn install```
3. clonar el archivo ```.env.template``` y renombrar a ```.env``
4. Cambiar la variables de entorno
5. levantar la base de datos
```
docker-compose up -d
```
6. Levantar: ```yarm start:dev```
